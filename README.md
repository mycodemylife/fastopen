# FastOpen - 桌面悬浮快捷启动工具

一款轻量级 Windows 桌面悬浮球应用，让你一键快速启动和切换常用程序。

## 功能特性

### 🎯 悬浮球

- 桌面常驻悬浮球，始终置顶显示
- 自由拖动到屏幕任意位置
- 靠近屏幕边缘自动吸附收缩，鼠标移入恢复
- 点击悬浮球展开快捷应用列表
- 根据悬浮球在屏幕的位置智能调整展开方向
- 鼠标悬停显示应用名称，点击即可启动
- 鼠标离开自动收起

### 🎨 外观自定义

**悬浮球样式**

| 样式 | 说明 |
|------|------|
| 默认 | 蓝色圆形 + 闪电图标 |
| 自定义图片 | 支持图片和 GIF 动图，自动适配大小 |
| 模拟时钟 | 实时走动的表盘，含时分秒针，可自定义颜色 |

**圆环线条**

- 自定义线条颜色和透明度
- 调整线条宽度（1px ~ 5px）
- 支持实线、虚线、点线三种样式

### 📋 快捷方式管理

- 拖拽 .exe 或 .lnk 文件添加应用
- 手动选择可执行文件添加
- 自动提取应用图标
- 支持修改名称、删除、拖拽排序
- 配置自动保存，重启后恢复

### 🔧 系统托盘

- 左键点击托盘图标打开配置窗口
- 右键菜单：打开配置 / 显示隐藏悬浮球 / 退出

## 系统要求

- Windows 10 / Windows 11
- 无需额外依赖，打包后独立运行
